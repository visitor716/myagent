# 清理文件资源管理器导航栏残留使用说明

## 问题原因

卸载百度网盘、iCloud 等软件后，注册表中仍有残留项，导致它们继续显示在文件资源管理器左侧导航栏。

## 使用方法

### 步骤 1：打开 PowerShell（管理员）

1. 按 `Win` 键，搜索 **PowerShell**
2. 右键点击 **Windows PowerShell** → **以管理员身份运行**

### 步骤 2：执行脚本

在 PowerShell 中输入以下命令：

```powershell
# 切换到脚本所在目录
cd \\wsl.localhost\\Ubuntu\\home\\zhanxp\\projects\\myagent\\Script

# 执行脚本
.\清理导航栏残留.py
```

或者直接拖放脚本到 PowerShell 窗口。

### 步骤 3：按提示操作

- 脚本会自动扫描并列出发现的残留项
- 清理完成后会自动重启资源管理器
- 按任意键退出

## 如果脚本无效

### 手动查找残留项

在 PowerShell（管理员）中运行：

```powershell
# 搜索相关注册表项
Get-ChildItem "Registry::HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace"

# 查看每个子项的默认值
Get-ChildItem "Registry::HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace" | ForEach-Object {
    $name = $_.GetValue("", "")
    $key = $_.Name.Split('\')[-1]
    Write-Host "[$key] = $name"
}
```

找到对应的名称后，记录其 `{}` 包裹的 GUID，然后删除：

```powershell
# 删除指定 GUID（将下面的 GUID 替换为实际的）
Remove-Item "Registry::HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace\{XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX}" -Recurse -Force
```

### 使用第三方工具

1. **Autoruns**（推荐）
   - 下载：https://docs.microsoft.com/en-us/sysinternals/downloads/autoruns
   - 运行后按 `Ctrl + F` 搜索 "Baidu" 或 "iCloud"
   - 取消勾选对应项

2. **ShellMenuView** / **ShellExView**
   - 可以管理右键菜单和 Shell 扩展

## 注意事项

- ⚠️ **必须以管理员身份运行**，否则无法删除注册表项
- ⚠️ 脚本会**自动重启资源管理器**，请确保没有未保存的工作
- ⚠️ 如果清理后仍有残留，尝试**注销或重启电脑**
