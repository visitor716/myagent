# Custom Skills

This directory is the only source of truth for user-managed custom skills.

## Rules

1. Create and edit custom skills here.
2. Keep one subdirectory per skill.
3. Each skill directory should include `SKILL.md`.
4. Each skill directory should include `.skill-source.json`.
5. Runtime paths under `~/.codex/skills/` and `~/.claude/skills/` should point back here.
